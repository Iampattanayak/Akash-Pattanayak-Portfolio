## ğŸš€ Quick Start

### Prerequisites

- Node.js 18+ 
- npm or bun

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Iampattanayak/akash-pattanayak-portfolio.git
   cd akash-pattanayak-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   bun install
   ```

3. **Start development server**
   ```bash
   npm run dev
   # or
   bun dev
   ```

4. **Open in browser**
   ```
   http://localhost:8082
   ```

### Build for Production

```bash
npm run build
```

The build output will be in the `dist/` folder.
